# rofi-extra-programs

## Descrição
Programa que lê funções/aliases do arquivo .bash_aliases e exibe elas no programa rofi.

## Observações
Essas funções/aliases são atalhos abreviados para executar programas com comandos específivos de maneira rápida, seja no terminal ou no próprio desktop, e agora com esse programa, será mais rápido a execução desses programas.

## Como compilar?
Antes de rodar é necessário compilar. Ele já será movido para o local correto ao término da execução do script.
```bash
./compilar_e_mover.sh
```

## Como rodar?
Apenas digite o nome do programa
```bash
rofi-extra-programs
```
